Estados de los Archivos en Git

1. ¿Cuál de los siguientes estados describe un archivo que ha sido cambiado, pero aún no está marcado para ser incluido en el próximo commit?
A. Preparado (Staged)
B. Modificado (Modified)
C. Confirmado (Committed)

2. Si un archivo está en el área de preparación (staging area) y luego se ejecuta el comando 'git commit', ¿a qué estado pasará el archivo en el repositorio local?
A. De "Preparado" a "Modificado".
B. De "Modificado" a "Confirmado".
C. De "Preparado" a "Confirmado".

3. ¿Qué opción del comando git commit permite añadir un mensaje directamente desde la línea de comandos?
A. git commit -m "mensaje"
B. git commit --add "mensaje"
C. git commit --msg "mensaje"
